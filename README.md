# 基于 FreeRTOS 的智能CAN通信与多维交互系统
- 项目描述：
将原有裸机系统重构为 FreeRTOS 多任务架构，实现按键触发发送、红外告警、CAN 回环通信及 OLED 实时监控功能
- 核心亮点与难点突破：
   - 异步解耦架构：利用 消息队列（Queue） 实现通信与 UI 显示任务解耦，消除 OLED 刷新对底层通信的阻塞
   - 中断性能优化：设计“ISR 快速读取 + 任务分发”机制，解决 CAN 高频接收导致的中断风暴与系统卡死问题
   - 多维状态机交互：结合任务优先级与 Tick 超时逻辑，实现呼吸灯、双速闪烁等报警状态的精准切换与自动复位
   - 工程调试与鲁棒性：通过 printf 重定向构建日志系统，定位并修复了堆栈溢出与初始化引起的 HardFault；引入空指针校验提升系统稳定性
